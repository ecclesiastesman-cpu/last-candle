# ПОСЛЕДНЯЯ СВЕЧА (Last Candle) — дизайн-документ

Тёмная оффлайн ARPG в духе Diablo для iPhone (Safari PWA) + десктоп. Язык: русский (строки внешние).

**Формула опыта:** игрок чувствует жадный азарт растущей силы посреди подступающего ужаса, потому что игра постоянно осыпает его эскалирующим лутом, уплотняет орды и оставляет светлым лишь круг его собственного света.

## Профиль
- Реальное время, непрерывное 2D top-down, один герой, vs system, процедурные подземелья + авторские акты, win/lose + бесконечный режим, соло, сессии 10–60 мин.
- Источники вовлечения: накопление (лут/уровни) + исполнение (уклонение, позиционирование).
- Платформы: mobile touch (основная, 390×844), desktop клавиатура (KeyW/A/S/D + JKLI скиллы), геймпад (стандартный маппинг). DPR cap 1.5, бюджет: 60 fps, ≤80 сущностей на экране, ≤300 частиц (пулы), 0 аллокаций в кадре.
- Сид RNG + фикс. таймстеп 60 Гц; логика отделена от рендера; сейвы в localStorage (авто).

## Метрики (заморожены до контента)
Тайл 64. Герой r=14, скорость 150 (волк 195, медведь 120). Мили-удар 52, снаряд 420 px/s, дальность 480. Экран телефона ~ 7×15 тайлов. Комната 8–16 тайлов, этаж 40×40.

## Классы (ресурс / 3 ветки × 6 талантов)
Очки: 1 талант/уровень + 5 характеристик (Сила/Ловк/Инт/Жив). Макс уровень 40. Респек за золото.
1. **Варвар** — Ярость (от ударов, распад вне боя). Оружие: Рассечение, Вихрь, Прыжок, Казнь / Крики: Боевой клич(бафф), Устрашение(страх), Клич крови(лич) / Стойкость: пассивы брони, шипы, Второе дыхание.
2. **Охотница** — Концентрация (реген в движении). Лук: Мультивыстрел, Пронзающая, Залп / Ловушки: Огненная, Морозная, Взрывная / Тени: Рывок, Ядовитый клинок, пассивы крита.
3. **Маг** — Мана. Огонь: Огн.шар, Стена огня, Метеор / Лёд: Ледяная стрела(замедл), Нова(заморозка), Осколки / Молния: Цепная, Телепорт, Гроза.
4. **Чернокнижник** — Души (с убийств). Проклятия: Гниение(DoT), Слабость, Жатва / Призыв: Скелеты(до 4, тинт mob_skeleton), Демон(тинт mob_imp) / Кровь: Кровавый шип(HP-кост), Вампиризм, Жертва.
5. **Друид** — Гнев природы; формы меняют панель. Волк: скорость, кровотечение, серия ударов / Медведь: танк, оглушение, рёв / Природа: Корни, Рой(DoT), Шторм, Исцеление. В формах шмот скрыт (канон D2).

## Лут
Слоты: оружие, щит/офф-хенд, шлем, доспех, перчатки, пояс, сапоги, амулет, 2 кольца. Видимые на кукле: оружие(6 визуалов), щит, шлем, доспех(2: латы/роба) + тинт по редкости + свечение (уникальные — янтарь, сеты — зелёный).
Редкости: обычн/магич(1–2 аффикса)/редкие(3–5)/уникальные(~15 фикс.)/сеты(2 сета по 3). Аффиксы (~36): +урон%, +атк.скор, крит шанс/урон, +скиллы класса, вампиризм HP, +HP/реген, броня, резисты(огонь/лёд/молн/яд), скорость, +золото/магия(MF), шипы, урон по нежити/демонам, -требования, +свет, сокеты нет (v2).
Экономика: золото — источник: дроп/продажа; сток: зелья, респек, гэмбл у торговца. Зелья HP пояс 4 заряда, реген у алтарей.

## Акты (процедурные этажи 3 + босс; хаб-лагерь между)
1. Проклятый погост/склеп — скелет, зомби, гуль → **Костяной Патриарх**
2. Чумные катакомбы — гуль, раздутый труп(взрыв), культист → **Мать Чумы**
3. Пыточные залы — гончая, культист, рыцарь ада → **Палач**
4. Врата Ада — бес, гончая, рыцарь → **Владыка Бездны**
Элитки: модификаторы (быстрый/огненный/ледяной/вампир/грозовой) — аура-тинт кодом. После акта 4 — бесконечные Порталы с масштабом и лучшим лутом. Сложности: Кошмар/Ад (NG+).
Хаб «Лагерь у последней свечи»: Скупщик (купля/продажа/гэмбл), Сундук, Алтарь талантов, Портал.

## Обучение (по одному паттерну)
Этаж 1.1: движение+атака → зелья; 1.2: скилл-кнопки + первый талант; 1.3: элитка; босс = экзамен. Далее новые типы врагов по одному.

## Аудио (процедурное, WebAudio — бюджет кредитов)
Тёмный дрон-эмбиент (осцилляторы+шум, случайные скрипы), SFX: удар, выстрел, заклинание, смерть монстра, лут, левел-ап, зелье, портал. Музыка боссов — ускоренный пульс.

## Доставка
PWA: manifest + sw precache, оффлайн полностью. Публикация: Higgsfield deploy (ссылка сразу) + отдельный репозиторий GitHub Pages. Кот (index.html корня artmore-card) неприкосновенен — игра живёт в /game и в отдельном репо.
